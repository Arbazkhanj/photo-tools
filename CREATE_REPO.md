# GitHub Repository Kaise Banayein

## Step 1: GitHub.com par jao
- https://github.com par login karo

## Step 2: New Repository
- Top right corner mein **"+"** icon pe click karo
- **"New repository"** select karo

## Step 3: Repository Details
```
Repository name: photo-tools
Description: Khan Jan Seva Kendra - Photo Tools
Visibility: Public
☑️ Add a README file (optional)
```

## Step 4: Create Repository
- **"Create repository"** button pe click karo

## Step 5: Repository URL Copy karo
- URL hogi: `https://github.com/Arbazkhanj/photo-tools`

## Step 6: Done!
- Ab main deploy kar dunga automatically
