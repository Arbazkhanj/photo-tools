# Khan Jan Seva Kendra - Photo Tools

## Deploy Instructions

### Step 1: Upload Files to GitHub
1. Go to: https://github.com/Arbazkhanj/photo-tools
2. Click **"Add file"** → **"Upload files"**
3. Upload all files from this folder:
   - index.html
   - CNAME
   - assets/ (folder)
   - images/ (folder)
   - .github/ (folder)
4. Click **"Commit changes"**

### Step 2: Enable GitHub Pages
1. Go to repository **Settings** → **Pages**
2. Source: Select **GitHub Actions**
3. Your site will be live at: https://khanjansevakendra.com
